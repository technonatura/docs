# Private API

Now let's play around with Private API! In this section you're gonna learn how to upload your sensor data to our database!

## Content

REST URL : https://technonatura.herokuapp.com/iot/add/data/ `POST`

content-type: `application/json`

- `iotAppToken`: Your IoT Cloud App Token, located in your app settings.
- `sensors`: Object of keys (sensors name), the data that will be uploaded.

<Callout emoji="ðŸ’¡">
	For example if you wanted to upload 23 integer data to your DHT11 Sensor then your content will be looked like this:
    ```
    {
        "iotAppToken": [paste your app token here],
        "sensors": {
            "DHT11": 23
        }
    }
    ```
</Callout>

## Return Values

- `message`: `string` data type, the message describes the response
- `status`: `string` data type, the status that describes the status of response; `success` | `warning` | `error` | `info`
