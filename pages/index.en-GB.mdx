## Public REST API

The following is a list of the Public APIs that we have created.

- **Users** API - show list of users, but only shows non sensitive data.
- **IoT Cloud Service Public API**
  - **Get Sensor**
  - **Get IoT Cloud App**

## Private API (The Cloud Service API)

- **IoT Cloud Service API**
  - **Upload Sensor Data to Database**

## GraphQL API

This type of API is still in the development stage, be patient.
